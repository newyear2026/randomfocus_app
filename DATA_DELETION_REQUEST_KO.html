데이터 삭제 안내

RandomFocus는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 페이지에서는 사용자 데이터 삭제 방법에 대해 안내합니다.

수집되는 데이터 유형

1. 기기 내부 저장소에 저장되는 데이터

  • 타이머 기록: 포커스 세션 기록 (시작 시간, 종료 시간, 세션 상태 등)
  • 스핀 사용 기록: 일일 스핀 사용 횟수 및 날짜 정보
  • 언어 설정: 선택한 앱 언어 설정
  • 앱 설정: 사용자가 변경한 앱 내 설정
중요: 이 데이터는 모두 사용자의 기기 내부 저장소에만 저장되며, 서버로 전송되지 않습니다.

2. 광고 관련 데이터

  • 광고 ID (Advertising ID): Google Mobile Ads를 통해 수집되는 광고 식별자
참고: 광고 ID는 Google Mobile Ads와 공유되며, 맞춤형 광고 제공을 위해 사용됩니다.

데이터 삭제 방법

모든 사용자 데이터는 기기 내부에만 저장되므로, 앱을 삭제하면 모든 데이터가 자동으로 삭제됩니다.

방법 1: 앱 삭제 (가장 간단한 방법)

1 Android 기기에서:
  • 설정 > 앱 > RandomFocus 선택
  • "삭제" 또는 "제거" 버튼 클릭
2 확인:

앱 삭제 시 다음 데이터가 모두 삭제됩니다:

  • 모든 타이머 기록
  • 스핀 사용 기록
  • 언어 설정
  • 모든 앱 설정
⚠️ 주의사항: 앱을 삭제하면 저장된 모든 데이터가 복구 불가능하게 삭제됩니다. 삭제하기 전에 중요한 기록이 있다면 백업을 고려하세요.

방법 2: 광고 ID 재설정 (광고 관련 데이터만)

광고 ID만 재설정하려면 다음 단계를 따르세요:

1 Android 기기에서:
  • 설정 > Google > 광고
  • "광고 ID 재설정" 선택
2 또는 광고 개인 맞춤 비활성화:
  • 설정 > Google > 광고
  • "광고 개인 맞춤" 옵션 비활성화

데이터 보관 기간

기기 내부 저장소 데이터

  • 보관 기간: 앱이 설치되어 있는 동안
  • 자동 삭제: 앱 삭제 시 즉시 삭제
  • 추가 보관 기간: 없음 (앱 삭제 시 즉시 완전 삭제)

광고 ID

  • 보관 기간: Google의 개인정보 처리방침에 따름
  • 삭제 방법: 기기 설정에서 광고 ID 재설정
  • 참고: Google의 개인정보 처리방침: https://policies.google.com/privacy

데이터 삭제 확인

데이터 삭제를 확인하려면:

  1. 앱을 삭제합니다
  2. 앱을 다시 설치합니다
  3. 이전 데이터가 모두 삭제되었는지 확인합니다
확인 사항:
  • 타이머 기록이 비어있음
  • 스핀 사용 기록이 초기화됨
  • 언어 설정이 기본값으로 돌아감

문의사항

데이터 삭제에 대한 추가 질문이나 문의사항이 있으시면 다음으로 연락해 주세요:

버전: 1.0.0
이메일: [개발자 이메일 주소를 입력하세요]
