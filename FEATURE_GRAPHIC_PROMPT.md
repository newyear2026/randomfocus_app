# Google Play Console Feature Graphic 이미지 생성 프롬프트

## 📐 이미지 사양

- **형식**: PNG 또는 JPEG
- **최대 용량**: 15MB
- **용도**: Google Play Console Feature Graphic

## 🎨 ChatGPT 프롬프트 (DALL·E 3 또는 이미지 생성 AI용)

### 프롬프트 1: 메인 버전 (영어)

```
Create a modern, minimalist feature graphic image for a mobile app called "RandomFocus". 

Design specifications:
- Dimensions: 1024px × 500px
- Style: Modern, clean, minimalist design with smooth gradients
- Color scheme: Primary color #6366F1 (deep purple), secondary colors include shades of purple (#8B5CF6), blue (#3B82F6), and white
- Background: Gradient from deep purple (#6366F1) to lighter purple/blue, flowing smoothly from top-left to bottom-right
- Main elements:
  * A colorful roulette wheel in the center-left area with 6 segments in different colors: orange (#FF9800), red (#EF4444), teal (#10B981), purple (#6366F1), blue (#3B82F6), and pink (#EC4899)
  * The roulette wheel should have numbers: 25, 30, 45, 50, 60, 90 (representing minutes)
  * A circular timer/progress ring on the right side, showing a focus timer with purple gradient
  * App name "RandomFocus" in large, bold, modern sans-serif font (white color with subtle shadow) positioned at the top or center
  * Tagline "Focus with Random Timer Sessions" in smaller text below the app name (white or light purple, semi-transparent)
- Visual effects: Soft shadows, subtle glow effects, clean lines
- No text overlay that would be hard to read
- Professional, app store quality design
- Ensure all text is readable and the design is balanced
```

## 🎯 프롬프트 사용 가이드

### ChatGPT/DALL·E 3 사용 시:

1. **기본 프롬프트 사용**
   - 프롬프트 1 또는 2를 복사하여 ChatGPT에 입력
   - "Create an image" 또는 "Generate an image"로 시작

2. **수정 요청**
   - 색상 조정: "Make the purple darker" 또는 "Use lighter blue"
   - 레이아웃 변경: "Move the roulette wheel to the left" 또는 "Make the timer larger"
   - 텍스트 변경: "Change the tagline to 'Boost Your Productivity'"

3. **최종 확인**
   - 크기가 정확히 1024×500px인지 확인
   - 텍스트가 읽기 쉬운지 확인
   - 색상이 앱의 브랜드 컬러와 일치하는지 확인
